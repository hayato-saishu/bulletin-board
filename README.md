# 概要

誰でも自由にスレッドを建てることができて、スレッド内で様々な情報を共有することができる。
また、コメントを残しているユーザーのプロフィールページに飛ぶことができる。
そこで、ゲーム内のIDなどを書き込んでおけば、フレンド登録することもできる。

# 開発環境

Rails 5.2.4.4
ruby 2.5.3
Server version: 5.5.62 MySQL Community Server (GPL)
